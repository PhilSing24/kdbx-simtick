# Notebooks

Interactive demos for `di.simtick` using PyKX.

## Setup

```bash
cd di/simtick
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
jupyter lab
```

## Available Notebooks

| Notebook | Description |
|----------|-------------|
| `simtickDemo.ipynb` | Load module, run simulation, visualize price and quantity |
